# Groq Smart Chatbot with Tavily Integration

A Python-based intelligent chatbot that leverages **Groq LLM** and the **Tavily search API** to answer user queries. The chatbot automatically decides whether a question requires **real-time information** and fetches relevant results from Tavily when needed.  

---

## Features

- Uses **Groq LLM** for natural language understanding and responses  
- Integrates **Tavily API** for fetching latest news and real-time updates  
- Automatically decides if a query requires external search  
- Logs activity and handles errors gracefully  
- Easy to run locally via Python or VS Code
